Mac:
====
brew update
brew install openssl
brew install pyenv
git clone https://github.com/yyuu/pyenv-virtualenv.git ~/.pyenv/plugins/pyenv-virtualenv

# Add to .bashrc:
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

PYTHON_CONFIGURE_OPTS="--enable-shared" CFLAGS="-I$(brew --prefix openssl)/include" LDFLAGS="-L$(brew --prefix openssl)/lib" pyenv install 3.5.4

# Then, in ~/dev/fman:
pyenv virtualenv 3.5.4 venv
pyenv activate venv

# Import developer certificate:
security unlock-keychain -p **PASSWORD**
security import "~/dev/fman/conf/mac/Private key.p12" -k ~/Library/Keychains/login.keychain -P "" -T /usr/bin/codesign

NSIS:
=====
Install version 2.51 on Windows.
Add to PATH.

Windows extra
=============
Add c:\Program Files (x86)\Windows Kits\10\bin\x86 to your PATH. This contains
signtool, which is used for codesigning.

Install Go 1.6.3 *32 bit*.

Linux (Ubuntu) extra
====================
sudo apt-get install bsdtar

This is required for creating the packages for pacman via fpm.
If the above fails, build https://www.libarchive.org/ from source and place its `bsdtar` on the PATH.

To be able to upload fman to the Arch User Repository during release, add ~/.ssh/id_rsa.pub to your account there.

Mac extra
=========
Import conf/mac/Developer Certificate.cer and .../Private key.p12 into KeyChain
