Build Qt 5.6.0:
=========
1) Download the 5.6.0 source tarball from the official home page.

Windows preparation:
 a) Install Visual Studio 2012 express
 b) Install http://www.activestate.com/activeperl
 c) Install Python
 d) Add gnuwin32/bin from Qt source dir, perl and Python to PATH
 e) Follow the instructions on http://doc.qt.io/qt-5/windows-building.html
    - You may have to install the DirectX SDK (9.29.1962.0)
 f) Add c:\Qt\5.6.0\qtbase\bin to your PATH.

OS X preparation:
 a) Edit the sources:
		qtbase/src/gui/kernel/qwindow.h,
		qtbase/src/gui/kernel/qwindow.cpp,
		qtbase/src/widgets/accessible/qaccessiblewidgetfactory_p.h:
			Surround all occurrences of QAccessibleInterface with
			#ifndef QT_NO_ACCESSIBILITY ... #endif
		qtwebengine/src/core/media_capture_devices_dispatcher.cpp:
			Add #include <QtCore/QCoreApplication>
		qtwebengine/src/webengine/api/qquickwebengineview_p_p.h:
			Add #include <QtGui/QColor>

Linux preparation:
 sudo apt-get install build-essential libxcb1 libxcb1-dev libx11-xcb1 libx11-xcb-dev libxcb-keysyms1 libxcb-keysyms1-dev libxcb-image0 libxcb-image0-dev libxcb-shm0 libxcb-shm0-dev libxcb-icccm4 libxcb-icccm4-dev libxcb-sync1 libxcb-sync-dev libxcb-xfixes0-dev libxrender-dev libxcb-shape0-dev libxcb-randr0-dev libxcb-render-util0 libxcb-render-util0-dev libxcb-glx0-dev libxcb-xinerama0-dev

2) Build & install
	./configure -release -opensource -no-accessibility -nomake examples -nomake tests
	make -j4 module-qtbase (on Windows use jom module-qtbase)
	sudo make module-qtbase-install_subtargets (on Windows use jom ...)

Install SIP:
============
 1) Download SIP 4.18 from https://www.riverbankcomputing.com/software/sip/download
 2) Unpack

On OS X:
 3) source venv/bin/activate
 4) python configure.py
 5) make
 6) make install

On Windows:
 3) open the Qt5 command console created according to http://doc.qt.io/qt-5/windows-building.html
 3) call venv/scripts/activate.bat
 4) python configure.py
 5) Replace `python34.lib` in siplib/Makefile by the absolute path c:\Python34\libs\python34.lib
 6) Replace `copy /y sipconfig.py...` in Makefile by its absolute path `copy /y c:\Users\Michael\Temp\sip-4.18\sipconfig.py ...`
 7) jom
 8) jom install

Install PyQT:
=============
 1) Download PyQt 5.6 from﻿https://www.riverbankcomputing.com/software/pyqt/download5
 2) Unpack

OS X:
 3) source venv/bin/activate
 4) python configure.py --qmake /usr/local/Qt-5.6.0/bin/qmake
 5) make
 6) make install

Windows:
 3) open the Qt5 command console
 4) python configure.py --qmake c:\Qt\5.6.0\qtbase\bin\qmake.exe --sip c:\Users\Michael\dev\fman\venv\sip.exe
 5) Search for `python34.lib` in all Makefile*'s and replace by absolute path c:\Python34\libs\python34.lib
 5) jom
 6) jom install

cx_Freeze:
==========

OS X:
-----
Execute the following while in the root project directory:

qt_lib_dir=/usr/local/Qt-5.6.0/lib
qt_lib_dir_str_len=25
for fname in $( find ${qt_lib_dir} -iname "*.framework" | cut -c ${qt_lib_dir_str_len}-9999 );
do
    ln -s ${qt_lib_dir}/$fname venv/lib/$fname
done

Linux:
------
sudo apt-get install lib32z1-dev
Download cx_Freeze.tar.gz from PyPI
source venv/bin/activate
python setup.py install
